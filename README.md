## auto_fuck_sj

这是一个 `agent skill` ，用来应付一些很简单的作业。

### 安装方法

复制该仓库链接给你的 Agent，然后告诉它：`帮我安装这个skill` 。

### 使用方法

- 在您的 `Agent` 中加载该 `skill` ，然后把记载有题目的文件、对应的demo、测试数据（可选）、`get_input_data.exe` 、`txt_compare.exe` 等需要的文件放到工作目录中。
- 对您的 `Agent` 说：帮我完成作业。

### 注意事项

- `Agent` 并不总能一次性产生符合要求的代码，可能需要您进行检查。
- 产生代码的风格**理论上**符合某位老师的要求，但是`Agent` 并不总能产生符合要求的代码，可能需要您进行检查。如果您不喜欢产生的代码的风格，可以让您的 `Agent` 更改 `skill` 的内容。

### 安全声明

**您可以使用该 `skill` 做任何您想做的事情。**
