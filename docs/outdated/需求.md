```
txt_compare（给定）用法
Usage: txt_compare.exe --file1 xxx --file2 xxx [ --trim none/left/rigth/all | --lineoffset -100..100 | --ignore_blank | --max_diff 0..100 | --display none/normal/detailed ]
                       必选项：
                          --file1 name1              : 指定要比较的第1个文件名（必选）
                          --file2 name2              : 指定要比较的第2个文件名（必选）

                       可选项：
                          --trim none/left/right/all     : 指定每行的空格/tab的忽略方式（无此项则默认为none）
                                none   : 每行均严格匹配
                                left   : 每行均忽略头部空格
                                right  : 每行均忽略尾部空格
                                all    : 每行均忽略头尾空格
                          --lineoffset n                 : 指定文件错位的行数（无此项则n为0）
                                取值[-100..100]，负数表示忽略file1的前n行，正数表示忽略file2的前n行
                          --ignore_blank                 : 忽略文件中--trim后的所有空行(无此项则不忽略)
                          --CR_CRLF_not_equal            : 不忽略Windows/Linux文件的换行符差异(无此项则忽略差异)
                          --max_diff m                   : 指定文件错位的行数（无此项则m为0）
                                取值[0..100]，表示满m个不同行则结束运行，不再进行后续的比较
                          --max_line x                   : 指定文件比较的最大行数（无此项则x为0）
                                取值[0..10000]，表示比较x行后则结束运行，不再进行后续的比较
                          --display none/normal/detailed : 指定显示的信息（无此项则默认为none）
                                none   : 仅一行匹配与否的提示
                                normal : 每个差异行给出差异信息
                                right  : 每个差异行给出更详细的差异信息

e.g.   txt_compare.exe --file1 my.txt --file2 std.txt
                           : 比较my.txt和std.txt，全部匹配，仅一行输出
       txt_compare.exe --file1 my.txt --file2 std.txt --trim all
                           : 比较my.txt和std.txt，每行均去除头尾空格，仅一行输出
       txt_compare.exe --file1 my.txt --file2 std.txt --lineoffset -2
                           : 比较my.txt和std.txt，忽略my.txt的前2行(即my.txt的第3行与std.txt的第1行进行比较，每行严格匹配)，仅一行输出
       txt_compare.exe --file1 my.txt --file2 std.txt --trim left --lineoffset 3
                           : 比较my.txt和std.txt，忽略std.txt的前3行(即my.txt的第1行与std.txt的第4行进行比较)，每行去除头部空格，仅一行输出
       txt_compare.exe --file1 my.txt --file2 std.txt --ignore_blank
                           : 比较my.txt和std.txt，忽略文件中的所有空行，仅一行输出
       txt_compare.exe --file1 my.txt --file2 std.txt --ignore_blank --trim right
                           : 比较my.txt和std.txt，忽略文件中去除头尾空格后的所有空行，仅一行输出
       txt_compare.exe --file1 my.txt --file2 std.txt --max_diff 3
                           : 比较my.txt和std.txt，全部匹配，有3行不匹配后结束运行，仅一行输出
       txt_compare.exe --file1 my.txt --file2 std.txt --max_line 7
                           : 比较my.txt和std.txt，全部匹配，仅比较前7行，仅一行输出
       txt_compare.exe --file1 my.txt --file2 std.txt --CR_CRLF_not_queal
                           : 比较my.txt和std.txt，全部匹配，不忽略Windows/Linux的文件差异，仅一行输出
       txt_compare.exe --file1 my.txt --file2 std.txt --display normal
                           : 比较my.txt和std.txt，全部匹配，不匹配的行显示信息
       txt_compare.exe --file1 my.txt --file2 std.txt --display detailed
                           : 比较my.txt和std.txt，全部匹配，不匹配的行显示详细信息


必须指定参数[--file1和--file2]
```
